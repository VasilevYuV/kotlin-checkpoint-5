# Консольный Task Manager (Kotlin)

Полноценное консольное приложение для управления задачами в стиле мини-backend сервиса.

## Реализованный функционал

- создание задач с автогенерацией `id`
- просмотр всех задач в виде таблицы
- поиск задач по нескольким условиям одновременно:
  - названию
  - приоритету
  - статусу выполнения
- редактирование задачи (пустой ввод сохраняет старое значение)
- удаление задачи по `id` с подтверждением
- сортировка по:
  - дате создания
  - приоритету
  - названию
  в порядке возрастания/убывания
- сохранение задач в JSON файл
- загрузка задач из JSON файла
- массовая отметка нескольких задач сразу (done / not done)
- обработка ошибок, защита от некорректного ввода
- логирование ошибок в файл `errors.log`

## Архитектура проекта

Проект построен по схеме уровня beginner: `Data + Repository + Menu`.

- `Task.kt` - `data class Task`
- `Priority.kt` - `enum class Priority` (1..5)
- `TaskRepository.kt` - слой работы с данными (CRUD, поиск, сортировка, save/load)
- `MenuController.kt` - слой взаимодействия с пользователем
- `AppLogger.kt` - логирование ошибок в `errors.log`
- `Main.kt` - только запуск приложения через контроллер меню

Прямых вызовов репозитория из `main()` нет: всё идёт через `MenuController`.

## Меню

```text
1 - Создать задачу
2 - Показать все задачи
3 - Найти задачу
4 - Редактировать задачу
5 - Удалить задачу
6 - Сортировка
7 - Сохранить в файл
8 - Загрузить из файла
9 - Отметить несколько задач
0 - Выход
```

## Формат данных

Используется JSON.

Пример одной задачи:

```json
{"id":3,"title":"Сделать проект","description":"Сделать модуль","priority":5,"isDone":false,"createdAt":"2025-02-10T12:11:45"}
```

## Запуск

2. Запустить `main()` из `src/Main.kt`.
